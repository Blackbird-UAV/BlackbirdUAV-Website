## How to Run Locally

1. Clone the repository:
   ```bash
   git clone [repository-url]
   ```
2. Navigate to the project folder:
   ```bash
   cd website
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```

The website should now be running locally at `http://localhost:3000`.
